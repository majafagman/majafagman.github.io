Asset project - Shell lounge chair made by Blanca

This project we were to create an asset for a school project. In the beginning I had a hard time deciding what I wanted to do but after some time I chose to create this shell lounge chair by Blanca.

The “shell” made of tree I created in Fusion 360, and the pillows I simulated and then sculpted in Blender. Uv-mapped and textured also took some time to create the perfect looking tree around the round corners.
